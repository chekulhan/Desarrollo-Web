## Usando Bucles en EJS

En EJS (Embedded JavaScript templates), puedes usar bucles para iterar sobre listas y generar contenido din√°mico.



```javascript

app.get('/numeros', (req, res) => {
  
  const nums = [1, 2, 3, 4];
  res.render('misnumeros', {nums});  // pagina misnumeros.ejs
});
```


```html
<ul>
  <% for (let index = 0; index < nums.length; index++) {  %>
    <li><%=index%></li>
  <% }  %>
 </ul> 


<ul>
  <% for (let i = 0; i < 5; i++) { %>
    <li>Elemento <%= i %></li>
  <% } %>
</ul>

```

Si pasamos un array desde el servidor, podemos recorrerlo con forEach():
```javascript
app.get('/productos', (req, res) => {
  const productos = ["Laptop", "Tel√©fono", "Tablet"];
  res.render('productos', { productos });
});
```

```html
<ul>
  <% productos.forEach(producto => { %>
    <li><%= producto %></li>
  <% }); %>
</ul>
```
